# Variational Autoencoder (VAE) - MNIST

This project implements a Variational Autoencoder using TensorFlow/Keras to generate handwritten digits.

## Features
- Encoder learns latent distribution (mean and variance)
- Reparameterization trick for sampling
- Decoder generates new digits
- Latent space visualization

## How to Run

```bash
pip install -r requirements.txt
python vae_mnist.py
